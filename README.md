# Copilot-Termux

GitHub Copilot CLI for Termux (Android aarch64).

Termux (Android aarch64) 向け GitHub Copilot CLI パッケージです。

## Status / 状態

> ⚠️ This package is currently in development / 現在開発中です

- version: (pending first release)
- package: \`@bash0816/copilot-termux\`

## Install / インストール（リリース後）

\`\`\`sh
# patchelf が必要
pkg install patchelf

npm install -g @bash0816/copilot-termux
copilot --version
\`\`\`

## Requirements / 必要環境

- Termux (Android aarch64)
- Node.js 18+（\`pkg install nodejs\`）
- patchelf（\`pkg install patchelf\`）
